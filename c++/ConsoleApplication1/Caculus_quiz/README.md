	1.求一個正整數n，他的二進制有多少個1

//計算步驟
//思路:令一個數字為2i+j和2i+0，
	2i+j是奇數，2i+0是偶數，對n不斷除以2，直到n=0
	11=10+1=5*2+1= 0000 1011，有3個1
	5 = 4+1=2*2+1= 0000 0101，有2個1
	12 = 12+0=6*2+0= 0000 1100，有2個1
	21 = 20+1=10*2+1= 0001 0101，有3個1
以11為例，11/2=5...1，5/2=2...1，2/2=1...0，1/2=0...1，總共有3個1
整除至0，餘數為1的次數就是二進制中1的個數

//---------------------------------------------------------
計算過程，重複次數為變量 = n / 2，直到n = 0，
	其中每次n % 2 = 1，count++，count就是二進制中1的個數
